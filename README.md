## ACERCA DEL PROYECTO
Traduciendo Con Sentido (TCS) es una aplicación inclusiva, para móvil que permite al público general y a personas con alguna discapacidad visual traducir lenguaje convencional a lenguaje braille, implementando la traducción de texto, documentos e imágenes.
